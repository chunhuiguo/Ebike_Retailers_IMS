# Inventory Management System (IMS) for E-bike Retailers

## Description
The IMS web application is designed for e-bike retailes and covers the following modules:
* Inventory Tracking (stock levels, locations)
* Order Management (sales/purchase orders, fulfillment)
* Warehouse Management (bin locations, transfers)
* Accounting Management
* Reporting & Analytics

## Architecture
The system is implemented with Java SSH (Struts + Sprint + Hibernate) architecture, JSP, and Microsoft SQL Server.
